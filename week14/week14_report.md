## 1. 项目概述

本项目实现了一个完整的手机遥控迷宫机器人系统。用户通过手机浏览器访问控制页面，使用触摸方向键遥控机器人在迷宫中移动；系统同时支持 BFS 和 A* 两种自动寻路算法，可一键完成迷宫探索。

## 2. 系统架构

```
手机浏览器 (index.html)
    ↕ WebSocket
Python 后端 (server.py)
    ├── maze.py       (迷宫生成: 递归回溯算法)
    └── explorer.py   (寻路算法: BFS + A*)
```

## 3. 核心功能

### 3.1 链路 (Link)
- **index.html**: 手机端触摸遥控界面，Canvas 实时渲染迷宫，D-Pad 方向键
- **server.py**: Tornado WebSocket 服务器，双向实时通信
- 网络架构: 手机浏览器 ↔ WebSocket ↔ Python 后端

### 3.2 迷宫 (Maze)
- **maze.py**: 递归回溯算法生成完美迷宫，支持自定义尺寸
- 墙壁数据结构: 每个 Cell 维护 N/S/E/W 四面墙壁
- 序列化接口: `to_dict()` 输出 JSON 供前端渲染

### 3.3 进阶 (Advanced)
- **explorer.py**: 
  - BFS 广度优先搜索 — 保证最短路径
  - A* 启发式搜索 — 曼哈顿距离启发函数
  - 自动探索器 — 增量探索 + 兴趣点记录
  - 性能基准测试 — 对比两种算法
